Detailed Description

A helper class to store mapping from triangles (represented by their int index) to components (represented by their int index) and cells (ulong), given following conditions:

a) Each triangle belongs to a cell, which is assigned a ulong coordinate, and to a component b) A component cannot span more cells - that is, if two triangles are in the same component, they must also be in the same cell
